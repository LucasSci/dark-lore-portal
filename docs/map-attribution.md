# Map Attribution

O atlas local usado em `/mapa` integra uma base cartografica derivada dos projetos:

- `witcher3map`: [https://github.com/witcher3map/witcher3map](https://github.com/witcher3map/witcher3map)
- `witcher3map-maps`: [https://github.com/witcher3map/witcher3map-maps/releases/tag/1.1](https://github.com/witcher3map/witcher3map-maps/releases/tag/1.1)

Uso neste projeto:

- consulta geografica dentro do produto
- apoio de navegacao para campanha e mesa
- contexto nao comercial

Diretriz de apresentacao:

- manter a atribuicao visivel no proprio atlas
- nao apresentar a base como autoria original do projeto
- preservar o atlas como referencia territorial, sem transformar a camada em marketing de terceiros
